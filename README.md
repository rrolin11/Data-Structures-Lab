# Estructuras de datos y algorítmos

Implementación de algunas estructuras de datos comunes y sus algorítmos más usados. Entre ellas se encuentran:
* Hash
* Graph
* AVL tree
* List
* Heap
* Stack
* Tuple
* Queue
* Priority queue
